# Raw Sources — 原始资料

> 🔒 此目录下所有文件不可变。LLM 只读不写。

## 目录结构

| 子目录 | 内容 | 示例 |
|--------|------|------|
| `papers/` | arXiv 论文、学术论文 | Memento-Skills 2603.18743 |
| `articles/` | 技术博文、Gist、演讲 | Karpathy LLM Wiki |
| `repos/` | GitHub 项目分析笔记 | MAF 源码结构 |
| `official/` | 官方文档摘录 | .NET 10、OpenTelemetry |
| `meetings/` | 会议/讨论记录 | 架构设计讨论 |
| `assets/` | 图片、图表等附件 | 架构图、截图 |

## 添加新资料

1. 将资料放入对应子目录
2. 文件名使用 `kebab-case.md`
3. 通知 LLM 执行 Ingest 操作
